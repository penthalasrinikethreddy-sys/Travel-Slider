Travel-Slider

Travel-Slider is a modern, mobile-first frontend application that showcases tourist destinations through an interactive swipe-based card interface. It delivers a smooth and engaging user experience by combining high-quality visuals, dynamic content, and seamless animations.

Built with React and Tailwind CSS, the app is fully responsive, lightweight, and designed for performance. It serves as an ideal UI solution for travel platforms, portfolio projects, and interactive frontend applications.
